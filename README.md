# 🚗 Car Price Predictor

A machine learning web application that predicts the price of a used car based on its features such as company, model, year, fuel type, and kilometers driven.

🔗 **Live App:** https://carpricepredictor-ayazg3iyb2xgdzy4jrnmf5.streamlit.app/  
📂 **Repository:** https://github.com/hammadulmustafa-albarvi/Car_price_predictor

---

## 📌 Problem Statement

Buying or selling a used car often involves guesswork and inconsistent pricing.  
This project aims to provide a **data-driven estimation** of car prices using machine learning.

---

## 🚀 Features

- Predict car prices instantly  
- Clean and interactive **Streamlit UI**  
- Handles multiple input features:
  - Company  
  - Model  
  - Year  
  - Fuel Type  
  - Kilometers Driven  
- Trained on real-world car dataset  
- End-to-end ML pipeline (data → model → deployment)

---

## 🧠 Tech Stack

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  
- Streamlit  

---

## ⚙️ How It Works

1. Data collection and cleaning  
2. Feature engineering  
3. Model training (Regression model)  
4. Model saved as `.pkl` file  
5. Streamlit app loads the model and predicts price  

---

## ▶️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/hammadulmustafa-albarvi/Car_price_predictor.git

# Navigate to project folder
cd Car_price_predictor

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
